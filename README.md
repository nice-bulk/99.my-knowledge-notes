## 📖 ファイル概要

### 01_Business-Mind — ビジネスマインド

| ファイル | 内容 |
| :--- | :--- |
| `customer-support-flow.md` | 顧客対応における認識合わせ・即答回避・出口戦略のフローチャート。プロフェッショナルな言い換えフレーズ集付き。 |
| `humor.md` | ユーモアをセンスではなくロジックとして習得するメソッド。「極端な具体数値」「逆転の肯定」「状況のズラし」の3パターンと参考文献まとめ。 |
| `panic-management.md` | 仕事でパニクった時に冷静さを取り戻すセルフマネジメント5ステップ。ジャーナリング → タスク整理 → スケジューリングの流れ。 |
| `positive-phrase-list.md` | 思考とチームの空気を最適化するポジティブ口癖ライブラリ。セルフブーストとチームコミュニケーションの2軸で整理。 |

### 03_Health-Management — 健康管理

| ファイル | 内容 |
| :--- | :--- |
| `exercise.md` | エンジニア向け1日25分・3回に分けた運動ルーティン。朝のストレッチ・昼の有酸素・夜のストレッチで座りすぎ・眼精疲労・姿勢を改善。 |
| `meal.md` | 集中力と栄養バランスを両立する1日の食事プラン。低GI・高タンパク・DHA/EPAを軸に、栄養達成率も整理。 |
| `happiness.md` | 幸福を支える4大物質（セロトニン・オキシトシン・ドーパミン・エンドルフィン）を状態に合わせてスイッチする幸福マネジメントマニュアル。 |
| `sleep.md` | 脳疲労と眼精疲労をリセットする睡眠・リカバリー習慣。4-7-8呼吸法・ハーブティー・ASMRを組み合わせた入眠ルーティン。 |

### 10_Framework — 発想フレームワーク

### 90_Prompts — AIプロンプト集

| ファイル | 内容 |
| :--- | :--- |
| `ai-doctor.md` | 体調不良時に一問一答形式で症状を整理するAIドクタープロンプト。病名推定・食事・市販薬・日別の過ごし方まで出力。 |
| `diary-assistant.md` | 毎日10分の一問一答式日記をサポートするAIアシスタント。9つの質問から日報レポート（感情分析・改善策・アドバイス）を自動生成。 |
| `idea-first-principles.md` | 第1原理思考でWebアプリを設計するプロンプト。常識の特定 → 基本真理への分解 → ゼロベース再構築の3ステップで革新的なアプリ案を生成。 |
| `idea-morphological.md` | 形態分析法でWebアプリのアイデアを生成するプロンプト。ターゲット・機能・シーン・技術の4軸を組み合わせてユニークなコンセプトを5案提案。 |
